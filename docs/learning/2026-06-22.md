# Learning Journal: 2026-06-22

## Q1: 为什么 NvimTree 没有配置 custom 过滤仍不显示 node_modules 和 dist

**Context**: `lua/insis/plugins/nvim-tree.lua`

**Key points**:

- `custom` 不是唯一过滤器，Git ignored 文件还有独立的 `filters.git_ignored` 过滤器。
- 本项目的旧配置 `git.ignore = true` 会被 NvimTree 迁移为 `filters.git_ignored = true`。
- `node_modules` 和 `dist` 通常在项目的 `.gitignore` 中，因此仍会被隐藏。
- 在 NvimTree 中按 `i` 可以临时切换 Git ignored 过滤器。

**Details**: [../knowledge/tooling/nvim-tree-filters.md](../knowledge/tooling/nvim-tree-filters.md)
